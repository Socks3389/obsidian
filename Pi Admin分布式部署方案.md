# 分布式部署方案 (宝塔面板版)

根据您的需求，我们将采用 **3台服务器** 进行分布式部署，并使用宝塔面板进行管理。

## 服务器规划
| 服务器角色   | 最低配置  | 部署内容                  | 说明                  |
| :------ | :---- | :-------------------- | :------------------ |
| **Web** | 2核 4G | 前端静态资源 (Nginx)        | 负责静态页面分发、反向代理、多域名接入 |
| **API** | 4核 8G | 后端服务 (NestJS + PM2)   | 负责业务逻辑、API 处理       |
| **DB**  | 4核 8G | MySQL 8.0 + Redis 7.0 | 负责数据存储，需开启远程访问      |
